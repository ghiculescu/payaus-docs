Look out! This page is for the technically minded. Head back to the home page if you're lost.

This site was generated using Jekyll, the static site generator. It’s hosted on Digital Ocean.

This site is open source! All the code is available on github, and contributions are always welcome.

We use a bunch of useful plugins contributed by other Jekyll users, as one as one written specifically for this site; lead_post.rb, which highlights the “primary” post in a category (and is used to generate the links on the home page). The tables of contents on posts use jekyll-table-of-contents, and the collapsible navigation on the sidebar uses nav.js (which should be branched out into a separate project one of these days).

PayAus is a fully integrated solution for rostering, reporting and timesheeting.
